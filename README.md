# Mining ERP System - Industrial Pro-Max

A high-performance, professional-grade Enterprise Resource Planning (ERP) system specifically designed for the mining industry. Built with a modern tech stack and "Industrial Pro-Max" design aesthetics.

## 🚀 Key Modules
- **Fleet Monitoring**: Real-time heavy equipment tracking.
- **Daily Shift Reports (DSR)**: Digital production logging.
- **Inventory Management**: Integrated stock tracking for Fuel, Explosives, and Spareparts.
- **Maintenance & Parts**: Preventive and breakdown maintenance scheduling.
- **Logistics & Fuel**: Automated fuel issuing and tank monitoring.
- **Invoicing & Billing**: Production-based billing system.
- **Safety & HSE**: Incident reporting and safety performance tracking.

## 🛠 Tech Stack
- **Frontend**: Next.js 14, Tailwind CSS, Lucide Icons.
- **Backend**: Next.js API Routes (Serverless).
- **Database**: MySQL with Connection Pooling.
- **Design**: Industrial Glassmorphism, 8-point grid system.

## 📦 Installation
1. Clone the repository: `git clone https://github.com/abadijayacopier/ERP.git`
2. Install dependencies: `npm install`
3. Configure `.env.local` with your MySQL credentials.
4. Import `database_schema.sql` into your MySQL server.
5. Run development server: `npm run dev`
